# Slack Agent Builder Challenge — Event Facts (verified 2026-07-11)

Source: https://slackhack.devpost.com/ (scraped 2026-07-11)

## Identity
- **Name:** Slack Agent Builder Challenge
- **Sponsor:** Salesforce, Inc. (Slack). **Administrator:** Devpost (Managed by Devpost).
- **Tagline:** "Whether it's your first agent or your fiftieth — Slack is where it ships."
- **Format:** Online, public. Themes: Beginner Friendly, Enterprise, Low/No Code.
- **Scale:** 4,160 registered participants (as of 2026-07-11). $42,000 cash pool ($74,400 total ARV).

## Dates (CRITICAL)
| Milestone | Time |

| **Submission deadline** | **Mon July 13, 2026, 5:00 PM PT (= Jul 14 00:00 UTC)** |

**As of July 11 (IST): ~2.5 days remain.**

## Qualifying technologies (must use ≥1)
1. **Slack AI capabilities** (Agent Builder, `slack create agent` CLI, HR/IT/Sales templates)
2. **MCP server integration**
3. **Real-Time Search (RTS) API**

## Tracks (must fit ≥1)
1. **New Slack Agent** — new agent that automates workflows / connects systems / surfaces insights. 1st $8k, 2nd $4k.
2. **Slack Agent for Good** — new agent addressing social impact (accessibility, education, economic opportunity, environment, public health, nonprofit ops). 1st $8k, 2nd $4k. Must explain impact in submission.
3. **Slack Agent for Organizations** — new OR significantly-updated existing agent **submitted to the Slack Marketplace before the deadline**; must follow all Marketplace guidelines **including installed in 5 active workspaces**; deployed in a production workspace (sandbox is for testing/judging only). 1st $8k (+30-min Slack product exec conversation + Stack Overflow podcast feature), 2nd $4k.

## Achievement prizes ($2k each; 1st/2nd place winners are NOT eligible — one prize per submission)
- Best UX
- Most Innovative Slack Agent
- **Best Technological Implementation** ← natural secondary target for an engineering-depth entry

## Eligibility
- 18+, resident of: Argentina, Australia, Belgium, Canada (ex-Quebec), France, Germany, **India**, Ireland, Japan, Luxembourg, Netherlands, New Zealand, South Africa, Spain, UK, USA.
- Teams up to 4. Multiple submissions allowed if substantially different.
- Government employees ineligible.
