# Stage-1 spike harnesses (historical)

These are the throwaway proof-of-feasibility scripts from Stage 1, kept for the
decision trail in `docs/decisions.md` (each spike's verdict froze a contract the
product was then built on). They use a fictional test fighter ("Dragan Petrovic")
and hardcoded fixtures BY DESIGN: they predate the database and are not product
code, are imported by nothing, and never run in production.
