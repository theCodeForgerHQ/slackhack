Internal audit spot-check

A compliance team needs to verify that 50 previously approved questionnaire answers are still accurate after a security policy update.

Manual baseline

An auditor manually re-checks each answer against Confluence, Jira, and Slack. At 0.5 hours per answer:

MetricValue
Answers to check50
Auditor hours25

With Asked & Answered

OutcomeValue
Answers still verified38
Flagged stale/contradicted8
Missing current evidence4
Auditor hours saved~18

Signed audit trail

For each answer, the agent generated a signed Agent Run Card showing the original approver, approval timestamp, citations, and a SHA-256 signature over the run metadata. The auditor could verify that no answer had been tampered with since approval.

“I used to spend days spot-checking questionnaire answers. Now I get a prioritized list of what changed and a signature proving the rest is intact.”
— Internal Auditor (illustrative)
This is a documented pilot scenario derived from the measured implementation. See IMPACT.md for the 2-week protocol that replaces it with customer data.
